# Vandal_Hack_2026_Quantimaniacs
Welcome! This is a project created during the 2026 University of Idaho Vandal Vault Hackathon.

### 1. Purpose
The purpose of the Buy v. Rent web application is to provide users with an easy resource to determine if, on average, it's better to rent or buy in their county. This statistic is determined using the popular "5% rule" - 
(Home Price × 0.05) ÷ 12 = Monthly Break-Even Rent
5% breakdown:
- Property Taxes (1% annually)
- Maintenance Costs (1% annually)
- Opportunity Cost (3% annually)

All data was collected using the FRED and HUD API. Using the average home price and the average rent price per month, an interactive map was created.

### 2. How to use
1. Ensure you have Python 3.13.3 downloaded on your machine
2. Download the ChatBox_Sandbox file onto your machine
3. Open it up in the IDE of your choice or in terminal
4. Run "python app.py" in terminal and then follow the localhost link provided in terminal
